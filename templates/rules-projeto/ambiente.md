# Ambiente de Trabalho

No inicio de cada sessao, identifique como o usuario esta usando o Claude:

- Claude Code no VS Code (extensao Anthropic no editor) — pode criar/editar arquivos e rodar comandos
- Claude Code no Terminal (comando `claude` no terminal) — pode criar/editar arquivos e rodar comandos
- Claude Desktop App (app com MCP configurado) — acessa arquivos via MCP
- Claude.ai no navegador (sem acesso a arquivos locais) — so texto

Adapte suas acoes ao ambiente. No Claude.ai web, nao tente criar arquivos locais.
